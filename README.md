## Experiment Overview
This lab demonstrates MQTT communication using an ESP32 microcontroller in the Wokwi simulator.
The ESP32 connects to WiFi, publishes messages to an MQTT broker, and subscribes to receive messages.

## Tools & Technologies
- ESP32 (Wokwi Simulator)
- MQTT Broker: test.mosquitto.org
- MQTTX (Client)
- Arduino IDE
- MQTT Protocol

## Features
- WiFi connection using ESP32
- MQTT publish & subscribe
- Real-time message monitoring
